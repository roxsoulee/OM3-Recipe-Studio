# OM-3 Recipe Studio v0.9 Public Beta

Independent OM SYSTEM OM-3 Recipe JPG / OM Workspace OES tool for Windows.

Internal build: v1.11.3f_AppIconReplacement

## What This Is

- A Windows onefile public beta based on independent OM SYSTEM OM-3 testing.
- A Recipe JPG and OM Workspace `.oes` helper for manual Color / Mono recipe workflows.
- Binary-only public beta distribution.

## Features

- Color Recipe editing with 12 Color Profile nodes.
- Mono Recipe editing.
- Open OM-3 Recipe JPG and backfill parameters.
- Export Recipe JPG.
- Import / export OM Workspace `.oes` recipe files.
- Load preview image.
- Display WB from loaded JPG / OES; set WB manually on camera.
- Custom output folder.
- Bilingual File / Help menu.
- Updated application / EXE icon.

## Important Notes

- This tool is not affiliated with, endorsed by, or supported by OM Digital Solutions, OM SYSTEM, Olympus, or OM Workspace.
- Tested for OM SYSTEM OM-3. Compatibility with other OM SYSTEM / Olympus cameras is not guaranteed.
- Preview is approximate. Final color may differ from camera capture or OM Workspace output.
- White Balance is displayed from JPG / OES when available, but it is not written into Recipe JPG. Set WB manually on camera.
- OES import/export is a convenience workflow and is not an official OM Workspace integration.
- The Windows EXE is unsigned. SmartScreen may show a warning.
